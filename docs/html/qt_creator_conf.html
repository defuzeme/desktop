QtCreator Configuration

Getting the Qt SDK

You need

  • Qt 4.7
  • QtCreator 2

Download the SDK containing both from http://qt.nokia.com/developer/qt-qtcreator-prerelease#download

QtCreator's options

In Outils > Options

  • Editeur de texte
    • Comportement
      • Uncheck "Inserer des espaces au lieu de tabulations"
      • Set "Taille de tabulation" and "taille de l'indentation" to the same values
  • C++
    • hpp/cpp
    • Select the license template (in client/helpers)
  • Designer
    • Generation de classe
      • Heritage multiple
      • Check "Prise en charge ..." and "Utilisez le nom ..."
  • Aide
    • Documentation
      • Click on Ajouter
      • Select the file client/docs/html/index.qch (see Using this doc in QtCreator for more details)
