Using this doc in QtCreator
Remarks:
If you followed the QtCreator Configuration page, it's already done !

Including the defuze.me client documentation into QtCreator? Yes you can!
It allows you to access the documentation directly into the editor.

Let's imagine:
You are in the Starter.hpp file and don't know what is the Cores class?
Just put the cursor on it and press F1, as you do for any Qt class, and the doc for Cores appears!

Isn't it awesome?

How-to

It's quite easy.

  • Go in Outils > Options
  • Go in Aide > Documentation
  • Click on Ajouter
  • Select the file client/docs/html/index.qch
