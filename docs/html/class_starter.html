Detailed Description

The start-up sequence.

It's displaying a splashscreen while loading CORES and static PLUGINS.

The sequence:

  • Set the applications general variables
  • Display the splashscreen
  • Create the CORES
  • Call init() on each
  • Create the PLUGINS
  • Call init() on each
  • Close the splashscreen
  • Give control to the main window
Attention:
CORES constructors are called with 1 arguments: the command line arguments.
